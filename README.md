# Flutter Modern Template

A modern Flutter template app with Material 3 design system and optimized build configuration.

## Features

- Material 3 design system
- Modern UI components
- Gradle 8.11.1 for optimal build performance
- Clean architecture ready
- Production-ready build configuration

## Getting Started

1. Clone this repository
2. Run `flutter pub get` to install dependencies
3. Run `flutter run` to start the app in development mode
4. Run `flutter build apk` to build a release APK

## Requirements

- Flutter SDK 3.5.0 or higher
- Dart SDK 3.5.0 or higher
- Android SDK for Android builds

## Build

The project is configured with Gradle 8.11.1 and includes optimized build settings for production releases.

```bash
flutter build apk --release
```
